# Project Overview

## Project Name

Job Portal System

## Objective

The Job Portal System is designed to connect employers with job seekers.

Employers can create and manage job postings while candidates can search and apply for suitable jobs.

## Main Users

- Guest
- Candidate
- Employer
- Administrator

## Main Features

- Candidate Registration
- Employer Registration
- Login
- Logout
- Search Jobs
- Apply Jobs
- Post Jobs
- Edit/Delete Jobs
- Manage Candidate Profile

## Testing Scope

The testing focuses on verifying all functional requirements described in the Software Requirement Specification.

Testing excludes:

- Performance Testing
- Security Testing
- Automation Testing
