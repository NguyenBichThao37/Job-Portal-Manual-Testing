# Job Portal Manual Testing Project

## Project Description

This repository contains manual testing artifacts for a Job Portal System.

The project focuses on verifying the major business functions of the application based on the Software Requirement Specification (SRS).

## Project Scope

The testing scope includes:

- Candidate Registration
- Employer Registration
- Login
- Logout
- Job Posting
- Edit/Delete Job Posting
- Apply for Jobs
- Candidate Profile
- Job Search

## Testing Types

- Functional Testing
- UI Testing
- Positive Testing
- Negative Testing
- Boundary Value Testing
- Validation Testing

## Deliverables

- Project Overview
- Test Scenarios
- Test Cases
- Bug Report

## Testing Tool

- Microsoft Excel
- SQL Server
- GitHub
